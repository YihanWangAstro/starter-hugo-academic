Secular is a fast solver for hierarchical secular dynamics, designed for studying long-term orbital evolution in triple and quadruple stellar/planetary systems.

```
$ secular --info
┌─────────────────────────────────────────────┐
│  Secular Dynamics Solver                    │
│  Hierarchical triples     ✓                 │
│  Quadruple systems        ✓                 │
│  Lidov-Kozai oscillations ✓                 │
│  Stability analysis       ✓                 │
│  Highly optimized         ✓                 │
└─────────────────────────────────────────────┘
```

**Key Features:**
- Solves orbit-averaged equations of motion for hierarchical systems
- Highly optimized for Lidov-Kozai oscillation studies
- Long-term stability limit analysis
- Applications: triple star systems, planet–binary interactions, compact object mergers
